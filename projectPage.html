DreamerGPT

Description:

The average person typically forgets their dream within minutes of waking up. Fortunately, DreamerGPT provides a solution to this problem. DreamerGPT is a unique full-stack web application that allows users to input their dreams and receive interpretations using OpenAI, providing insight and clarity for the user.
To use DreamerGPT, users can either log in or sign up using session authentication. Once logged in, they can navigate to their "Journal Page" and begin logging their dreams as they occur. The journal page displays all previously logged dreams for easy reference.

RecipeHub

Description:

RecipeHub is a website that provides users with the option to find a random recipe or search for a recipe based on their criteria. After the search, the website presents a list of local Charlotte grocery stores where users can purchase the required ingredients. To retrieve recipe data for over 300 different recipes, the website employs multiple API calls to TheMealDB. Additionally, all searched recipes are saved to local storage, enabling users to access them again when they return to the webpage.

TechBlog

Description:

TechBlog is a Full-Stack CRUD web application that primarily utilizes Express.JS, Handlebars.JS, and Sequelize. It offers signed-in users a platform to post anything related to tech. Users are prompted to either login or sign up and can access their dashboard after doing so. Within the dashboard, users can view all of their posts and comments and have the option to update or delete them as needed.

Simon Game

Description:

This is a recreation of the classic 90's game, Simon. The objective of the game is to remember a pattern that is randomly generated for the user for as long as possible, or until they reach the final level. Upon completion of the game, the level achieved and the time taken to complete it are saved to local storage, allowing the user to retrieve the information at a later time.

Coding Quiz

Coding Quiz is an interactive quiz that covers a variety of coding-related topics. Users are given 1 minute to answer 10 questions. If the user selects an incorrect answer, 10 seconds are deducted from the game clock. The game concludes when the user completes the final question or when time runs out. User scores are saved to local storage and can be retrieved at any time.
